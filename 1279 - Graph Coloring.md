# 1279 - Graph Coloring
##  Problem Idea:
**color(v) = color(u1) + color(u2) + ... + color(um) (modulo K)**  
There are n equation for  each  **0**  ≤  **v**  ≤  **n** .
We can find the value of each v by using Gauss-Jordan elimination.
